# 作品集编辑拼贴风改版设计

## 目标

将现有暖色纸张风作品集整体改造成大胆但专业的编辑拼贴风，提高招聘方浏览时的第一眼记忆度，同时保持内容清晰、交互稳定和移动端可用。

## 已确认方向

- 风格：编辑拼贴风
- 配色：暖调流行色
- 改造范围：整站视觉系统与适度版面重排
- 落地方式：编辑网格保证阅读，重点区域使用局部拼贴
- 内容范围：保留现有文案、作品图片与核心功能

## 视觉系统

### 色彩

- 暖白纸张色作为主背景，替代现有粉绿光晕渐变。
- 珊瑚粉作为主要强调色。
- 亮紫与明黄作为局部辅助色，控制使用面积，避免影响阅读。
- 深黑用于标题、正文、边框和硬阴影，建立清晰的信息层级。

### 字体与层级

- 大标题使用具有杂志感的衬线字体，形成封面式视觉重心。
- 正文、标签和功能按钮使用现代无衬线字体，保证中文可读性。
- 通过超大标题、小号期刊信息、编号和标签制造编辑排版对比。

### 图形语言

- 使用错位色块、剪纸几何、细颗粒纹理、期刊编号和贴纸标签。
- 装饰元素不覆盖正文与关键操作，不与作品图片争夺视觉焦点。
- 作品图片保持原色，通过不规则边框、轻微旋转和硬阴影形成拼贴感。

### 动效

- 页面进入采用短促滑入、翻页和贴纸弹出效果。
- 保留现有关键词互动、轮播、拖拽画廊和图片查看器。
- 避免持续晃动或大幅旋转；系统启用减少动态时关闭非必要动画。

## 页面结构

### 导航

导航改为暖白杂志栏，使用黑色细边框。展开菜单呈现彩色目录卡片，现有锚点与联系方式入口保持不变。

### 首页

首页重排为不对称杂志封面：姓名和核心关键词作为主标题区，头像处理为剪贴照片，并加入期刊编号、色块与少量几何装饰。保留关键词物理互动和头像点击还原功能。

### HR 流程

采用双栏编辑网格。流程图和说明形成类似杂志跨页的组合，标签系统改为更鲜明的编辑索引样式。

### 作品区

保留交错排列和全部作品内容。每个项目增加大编号、分类标签、硬阴影及轻微错位；图片仍可打开完整图，轮播行为保持不变。

### 弧形画廊

保留拖拽与键盘操作。卡片外观改为纸片拼贴，边缘和标签与新视觉系统一致。

### 关于与联系

关于区作为编辑附录，使用清晰网格组织能力信息；联系区作为彩色封底，用高对比色块强化行动入口。

## 技术边界

- 继续使用现有 `index.html`、`styles.css` 和 `script.js` 静态结构，不引入框架或构建工具。
- 优先通过 CSS 变量、伪元素和现有 DOM 类名完成改造。
- 只在现有结构无法表达新布局时做少量语义化 HTML 调整。
- JavaScript 仅调整与新动效或布局相关的参数，不重写已稳定的交互逻辑。

## 响应式与无障碍

- 桌面端采用不对称编辑网格；移动端回落为单列，装饰元素减少并限制在内容边界内。
- 页面不得出现横向溢出。
- 保持文字与背景对比度、键盘焦点样式、按钮语义和图片替代文本。
- `prefers-reduced-motion` 下关闭非必要位移、旋转和弹出动画。

## 验收标准

- 整站呈现统一的暖调编辑拼贴风，首页与作品区有明显视觉变化。
- 现有文字、图片和作品顺序未被意外修改。
- 导航、关键词互动、头像还原、轮播、画廊拖拽和图片弹窗正常工作。
- 常见桌面和手机宽度下无横向滚动或内容遮挡。
- 页面加载后无控制台错误，图片资源均能正常显示。
- 减少动态模式生效，键盘操作和焦点可见性保持正常。

## 非目标

- 不新增作品、修改个人信息或重写文案。
- 不更换图片资产。
- 不引入后台、数据库、部署流程或第三方 UI 框架。
